Hostel Maintenance Portal

📌 Description

This system allows students to report maintenance issues and track their status. Maintenance staff and supervisors can manage and resolve requests.

🚀 Features

* User authentication (Student, Staff, Supervisor)
* Submit maintenance requests
* Track request status
* Admin dashboard

⚙️ Setup Instructions

1. Install XAMPP
2. Import `database.sql` into phpMyAdmin
3. Copy project folder into `htdocs`
4. Run: http://localhost/hostel-maintenance-portal
